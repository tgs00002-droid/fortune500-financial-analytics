# Fortune 500 Financial Analytics Platform

## Overview
This project analyzes the top 500 Fortune-ranked U.S. companies using official Fortune data, enriched with financial and market intelligence.

The goal of this project is to showcase real-world financial analytics skills by combining:
- Company revenue and profit rankings
- Market performance data
- Visual storytelling and business insights

This project is built entirely in Python and designed to mirror the type of internal analytics tools used by strategy, finance, and analytics teams at large companies.

---

## Data Sources
- Fortune 500 company ranking data (latest available year)
- Public market data via financial APIs (stock prices, market cap, returns)
- Company branding assets (logos) for visualization

---

## Key Features (In Progress)
- Top 500 company overview with rankings
- Revenue and profit analysis
- Industry-level comparisons
- Company-level financial drilldowns
- Visual dashboard with company logos

---

## Tech Stack
